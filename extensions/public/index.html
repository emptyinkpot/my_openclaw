🗄️ MySQL 数据库字段说明

📚 works 表 - 作品表

字段名 类型 说明
id INT AUTO_INCREMENT PRIMARY KEY 作品唯一标识
title VARCHAR(255) 作品标题
status VARCHAR(50) DEFAULT 'outline' 作品状态（outline/pending/audited/published）
platform VARCHAR(100) 发布平台
total_words INT DEFAULT 0 总字数
created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP 创建时间
updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP 更新时间

📑 chapters 表 - 章节表

字段名 类型 说明
id INT AUTO_INCREMENT PRIMARY KEY 章节唯一标识
work_id INT 所属作品ID
chapter_number INT 章节序号
title VARCHAR(255) 章节标题
content TEXT 章节内容
word_count INT DEFAULT 0 字数
status VARCHAR(50) DEFAULT 'pending' 章节状态（outline/pending/audited/published）
audit_status VARCHAR(50) DEFAULT 'pending' 审核状态（pending/reviewing/passed/failed）
audit_issues JSON 审核问题列表
suggested_action VARCHAR(50) DEFAULT 'none' 建议操作（auto_fix/manual/none）
created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP 创建时间
updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP 更新时间

👥 characters 表 - 角色表

字段名 类型 说明
id INT AUTO_INCREMENT PRIMARY KEY 角色唯一标识
work_id INT 所属作品ID
name VARCHAR(255) 角色名称
description TEXT 角色描述
created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP 创建时间
