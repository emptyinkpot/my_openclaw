## 一、主站登录恢复

### 操作信息

| 要素 | 内容 |
|------|------|
| **操作名称** | 主站登录恢复 |
| **前置操作** | 无（入口操作） |
| **URL** | `https://www.coze.cn/` |
| **入口UI** | 直接访问URL |
| **页面UI** | 主页，顶部导航栏，可能有登录弹窗 |
| **作用** | 设置登录状态，为后续操作做准备 |
| **通往** | 扣子编程、个人中心、主站功能 |
| **验证方式** | localStorage 数量 > 10，页面显示用户信息 |

### URL

```
https://www.coze.cn/
```

### 页面UI

```
┌─────────────────────────────────────────────────────────┐
│  [Logo] 扣子                    [登录] [注册]           │  ← 未登录
├─────────────────────────────────────────────────────────┤
│                    主页内容                             │
└─────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────┐
│  [Logo] 扣子     [功能列表]          [用户头像]          │  ← 已登录
├─────────────────────────────────────────────────────────┤
│                    主页内容                             │
└─────────────────────────────────────────────────────────┘
```

### 关键代码

```javascript
const backupFile = '/workspace/projects/cookies-accounts/coze-account-1-full.json';
const data = JSON.parse(fs.readFileSync(backupFile, 'utf-8'));

// 1. 访问主站
await page.goto('https://www.coze.cn/', { timeout: 30000 });
await page.waitForTimeout(2000);

// 2. 设置 localStorage
await page.evaluate((items) => {
  for (const [k, v] of Object.entries(items)) {
    try { localStorage.setItem(k, v); } catch (e) {}
  }
}, data.localStorage || {});

// 3. 设置 cookies
await context.addCookies(data.cookies);
```

### 注意事项

- ⚠️ **必须先访问主站**，才能跳转子站点
- localStorage 和 cookies 都需要设置
- 等待时间至少 2 秒确保设置生效

https://code.coze.cn/home

// 前置：已完成主站登录恢复

### 注意事项

- ⚠️ **必须先完成主站登录恢复**

## 三、进入项目管理

### 操作信息

| 要素 | 内容 |
|------|------|
| **操作名称** | 进入项目管理 |
| **前置操作** | ⭐ 主站登录恢复 → ⭐ 进入扣子编程 |
| **URL** | `https://code.coze.cn/w/{workspace_id}/projects` |
| **入口UI** | 侧边栏"项目" → "项目管理" |
| **页面UI** | 项目卡片列表，筛选栏，搜索框，新建按钮 |
| **作用** | 查看和管理所有项目 |
| **通往** | 具体项目详情、新建项目 |
| **验证方式** | URL 包含 `/projects`，显示项目卡片列表 |

### URL

```
https://code.coze.cn/w/{workspace_id}/projects

示例：
https://code.coze.cn/w/7587425486419378228/projects
```

> ⚠️ URL 中的 `{workspace_id}` 是工作空间ID，每个用户不同

### 关键代码

```javascript
// 前置：已完成主站登录恢复 + 进入扣子编程

// 方式1: 直接跳转URL（推荐）
await page.goto('https://code.coze.cn/w/7587425486419378228/projects', { timeout: 30000 });

// 方式2: 点击侧边栏
await page.evaluate(() => {
  const sidebar = document.querySelector('[class*="sidebar"]');
  const items = sidebar?.querySelectorAll('a, button, [class*="item"]');
  for (const item of items || []) {
    if ((item.innerText || '').trim() === '项目管理') {
      item.click();
      break;
    }
  }
});
```

- 可以直接跳转URL，无需逐步点击

## 六、常见问题

### Q: 为什么必须先访问主站？

A: 扣子编程 (`code.coze.cn`) 和主站 (`www.coze.cn`) 是不同域名，cookies 需要在主站设置后才能共享到子站点。

### Q: workspace_id 从哪里获取？

A: 可以从 URL 中获取，或者通过 API 获取。当前工作空间ID: `7587425486419378228`

### Q: 直接跳转URL和点击菜单有什么区别？

A: 效果相同，直接跳转URL更快，但需要知道正确的URL格式。
