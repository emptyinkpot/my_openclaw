## 经验4: 数据清洗和去重

**状态**: ✅ 已验证

**问题**: 文档中章节号重复（如第1章出现多次）、标题带冒号

**正确做法**:
```python
# 去重
seen = set()
unique = []
for ch in chapters:
    if ch['number'] not in seen:
        seen.add(ch['number'])
        unique.append(ch)

# 清理标题
title = title.replace(/^[：:]/, '').strip()

# 过滤异常
if 1 <= ch['number'] <= 200:  # 只保留有效范围
```

**适用场景**: 数据预处理

## 经验5: 导入流程标准化

**状态**: ✅ 已验证

**标准流程**:
```
1. 确认目标作品ID
2. 解析文档（段落+表格）
3. 提取章节（去重、排序）
4. 分离正文/大纲
5. 标记空章节（字数<300）
6. 计算卷号
7. 清空旧数据（事务保护）
8. 分批插入
9. 验证统计（各表count）
```

**验证查询**:
```javascript
// 验证数据完整性
const [chCount] = await conn.execute(
  'SELECT COUNT(*) as c FROM chapters WHERE work_id = ?', [workId]
);
const [olCount] = await conn.execute(
  'SELECT COUNT(*) as c FROM chapter_outlines WHERE work_id = ?', [workId]
);
console.log(`chapters: ${chCount[0].c}, outlines: ${olCount[0].c}`);
```

**适用场景**: 所有数据导入任务

## 常见错误

| 错误 | 原因 | 解决 |
|-----|------|------|
| 章节数不对 | 没有去重 | 使用seen=set() |
| 内容为空 | 只读了段落没读表格 | 检查doc.tables |
| 卷号错误 | 边界值判断错误 | 用>=和<= |
| 标题有冒号 | 没有清理 | replace(/^[：:]/, '') |
