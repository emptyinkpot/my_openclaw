# 经验积累中心功能增强 - 笔记合并和详情查看

## Summary
经验积累中心功能增强 - 笔记合并和详情查看

## Content
## 功能概述

在经验积累中心的笔记功能中添加了两个新功能：

### 1. 笔记合并功能
- 在笔记卡片上添加了复选框，支持多选笔记
- 添加了"合并选中笔记"按钮（仅在选中2条或以上笔记时显示）
- 合并时会自动：
  - 生成合并后的标题（原标题用 + 连接）
  - 合并所有笔记内容（用分隔线分隔）
  - 合并所有标签（去重）
  - 继承第一个笔记的分类
- 支持在保存前编辑合并后的标题、内容、分类和标签

### 2. 笔记详情功能
- 点击笔记的标题、图标或内容区域会打开详情弹窗
- 详情弹窗显示：
  - 完整标题
  - 分类和更新时间
  - 所有标签
  - 完整内容（支持换行和格式化）

## 修改文件
- `extensions/novel-manager/public/experience.html`

## UI 改进
- 笔记卡片选中时有蓝色边框和渐变背景高亮
- 合并按钮在选中足够笔记时自动显示
- 详情弹窗支持更大的宽度（680px）来显示完整内容
