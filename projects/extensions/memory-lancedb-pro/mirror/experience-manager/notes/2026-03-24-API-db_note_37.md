# 经验积累中心空白页排查

## 现象

经验积累中心看起来是空的，records 不显示。

## 真正原因

1. 页面请求经验接口时没有带对的 bootstrap token，接口直接返回 401。
2. 旧页面还在使用错误的 API 前缀或写死的 token。
3. 搜索/筛选逻辑如果从已经过滤过的数组继续过滤，会让页面一直像“没数据”。

## 修复

- 统一读取 `openclaw.control.bootstrap.token.v1`。
- 统一 API 前缀为 `/api/experience`。
- 搜索和筛选都从完整数据源 `allExpData` 重新计算。
- 用真实接口返回和真实 DOM 两层验证。

## 验证标准

- `GET /api/experience/stats` 返回 200。
- `GET /api/experience/records` 返回 200。
- 页面能渲染出 records，不再只是空状态。
- 刷新或重新搜索后，列表仍然稳定。
