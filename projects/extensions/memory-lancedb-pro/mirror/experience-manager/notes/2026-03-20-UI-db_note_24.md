# 前端UI和顶部导航栏问题解释

用户询问：
1. 所有的前端UI在哪啊？
2. 怎么经验的顶部导航栏和其他的不一样？

---

## 1️⃣ 所有的前端 UI 在哪啊？

OpenClaw 的 Control UI（管理后台）不在你的项目目录里，而是在 OpenClaw 的系统安装目录里！

看你的 openclaw.json 配置：
```json
"gateway": {
  "controlUi": {
    "root": "/usr/lib/node_modules/openclaw/dist/control-ui"
  }
}
```

**位置：**
```
/usr/lib/node_modules/openclaw/dist/control-ui/
```

---

## 2️⃣ 怎么经验的顶部导航栏和其他的不一样？

你说的"经验的顶部导航栏"，可能是指：

### 可能的情况：

**情况 A：你看到的是不同的 UI**
- OpenClaw 的 Control UI 是统一的
- 但可能你看到的是：
  - Control UI（管理后台）
  - 或者是某个插件的 UI（比如 novel-manager、experience-manager）

**情况 B：插件自己的 UI**
- 你的项目里有 novel-manager、experience-manager 这些插件
- 这些插件可能有自己的 UI 界面
- 它们的导航栏可能和 Control UI 不一样

---

## 📂 你的项目里的 UI 相关文件夹

看你的项目目录：
```
/workspace/projects/
├── extensions/apps/
│   ├── novel-manager/          # 小说管理插件（可能有自己的 UI）
│   ├── experience-manager/     # 经验管理插件（可能有自己的 UI）
│   ├── memory-lancedb-pro/
│   └── feishu-openclaw-plugin/
```
