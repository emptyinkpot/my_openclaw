# 修复点击设置按钮出现旧导航栏重叠的问题

## Summary
修复点击设置按钮出现旧导航栏重叠的问题

## Problem
在非小说管理页面点击"设置"按钮时，没有弹出设置弹窗，反而出现旧导航栏重叠在新导航栏上。通过浏览器控制台调试发现，fetch("/settings-modal.html") 错误地返回了原生界面的 HTML（里面包含旧导航栏），而不是我们想要的设置弹窗内容。

## Solution
将 settings-modal.html 的完整内容（包括样式、HTML结构和JavaScript函数）直接硬编码到 nav-bar.html 中，完全避免使用 fetch 动态加载。这样无论在哪个页面，点击"设置"按钮都能直接使用本地嵌入的设置弹窗，不再依赖网络请求和路由匹配。
