# 开发经验积累与学习系统

## Summary
用户要求创建自我学习和经验积累模块，记录每次回答如何运用经验以及如何积累经验。系统包括经验类型、难度等级、解决方案等维度。

## Problem
你不是有自我学习和经验积累模块吗，开一个ui入口显示积累了多少经验

## Solution
1. 设计经验积累数据模型
2. 创建ExperienceRepository类使用JSON存储
3. 添加API路由
4. 在UI顶部导航添加入口
5. 创建经验面板页面
6. 添加详情和添加记录的模态框

## Applied
- 模块化设计
- 前后端分离
- JSON存储

## Gained
- 经验系统设计
- 数据可视化
- 等级进度计算
