# 流水线使用说明

## 操作步骤

### 1. 扫描作品
点击「📡 扫描作品」按钮，系统会自动扫描所有选中账号的番茄作品。

### 2. 选择作品
在扫描出的作品列表中，**点击一个作品卡片**进行选择。
- 选中的作品会有蓝色边框
- 界面会提示 "📖 已选择: xxx"

### 3. 配置发布参数
- **试运行模式**: 勾选后只模拟发布，不真正提交
- **后台运行**: 勾选后浏览器无头运行
- **跳过审核**: 勾选后跳过审核状态检查
- **章节范围**: 可指定发布特定范围的章节

### 4. 启动流水线
点击「▶ 启动流水线」按钮开始发布。

## 故障排查

### 问题1: 进度条没有变化

**检查步骤**:
1. 打开浏览器开发者工具 (F12)
2. 切换到 Console 标签
3. 查看是否有错误信息
4. 查看 Network 标签，确认 SSE 连接是否建立

**可能原因**:
- 浏览器缓存了旧版本页面 → 强制刷新 (Ctrl+Shift+R)
- 没有选择作品 → 先点击作品卡片
- SSE 连接失败 → 检查网络

### 问题2: 提示"请先选择一个作品"

**解决方法**: 在作品列表中点击一个作品卡片进行选择。

### 问题3: SSE 连接断开

**检查步骤**:
1. 打开浏览器控制台
2. 查看 `[SSE]` 开头的日志
3. 确认连接状态

### 问题4: 发布失败

**检查步骤**:
1. 查看控制台错误信息
2. 确认 Cookie 是否过期
3. 尝试关闭「后台运行」模式查看实际页面

## 日志查看

### 浏览器控制台
- `[SSE] 连接:` - SSE 连接地址
- `[SSE] 连接已建立` - 连接成功
- `[SSE] 收到消息:` - 进度更新
- `[PipelineUI] 更新进度:` - UI 更新

### 服务端日志
```bash
tail -f /app/work/logs/bypass/dev.log | grep -E "Pipeline|SSE"
```
